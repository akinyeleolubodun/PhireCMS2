REQUIREMENTS
------------
The basic requirements for Phire CMS 2.0.0 are as follows:

* PHP 5.3.0+
* Apache 2+, IIS 7+, or any web server with URL rewrite support
* Supported Databases:
    - MySQL 5.0+
    - PostgreSQL 9.0+
    - SQLite 3+
